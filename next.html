Next Steps

Software

Our goal with out software moving forward is to create a more complex triangulation system. This means adding obstacles that the triangulation system will automatically avoid. Along with that, we would like to update our user input interface in order to make it more intuitive and visual appealing.

Electronics

For our physical electronics, there isn't that much that we'd like to do. However, one thing that is part of our plan moving forward is to move all of our software onto a raspberry pi. This is in order to make our rovers be able to function away from a computer, by having a smaller computer attached.

3D Model

While our CAD model is pretty much done, we would like to attach all of our electronics and software to our phsyical model, merging all of our different parts together. If this requires updating the model, then that will be done as well.

Simulation

As we go forward, we will continue to work on our simulation, adding new terrain and testing new environments. One specific things that that includes is testing with different levels of gravity, which will allow us to get a more accurate impression of how the rovers will actually work.
